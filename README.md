# Trabajo-final

### ¿Para qué es esta propuesta?

##### Esta propuesta tiene como propósito construir un portafolio profesional que exponga mis trabajos de diseño industrial con claridad y elegancia, y al mismo tiempo proyecte una futura especialización en diseño interior y arquitectura mediterránea minimalista. El sitio mostrará mis proyectos con una estética pulida, sobria y con un toque cálido mediterráneo.
